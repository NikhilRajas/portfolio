# Nikhil Rajas - Portfolio Website

### Overview
Professional portfolio website showcasing Nikhil Rajas' expertise as an AWS Data Engineer with experience in cloud migration, data pipelines, and big data technologies.

### Features
- Responsive dark theme design with neon accent
- Animated page transitions
- Showcase of AWS and Azure certifications
- Detailed experience with Toyota Motors and Merck Pharmaceuticals
- Contact form and social links
- Mobile-friendly responsive design

### Tech Stack
- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter, Dancing Script)

### Setup
1. Clone the repository
2. Add Nikhil's photo to `assets/img/avatar.jpg`
3. Upload resume to `assets/Nikhil_Rajas_Resume.pdf`
4. Update LinkedIn and GitHub URLs in `contact.html`
5. Open `index.html` in a browser
